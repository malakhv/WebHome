Yandex

Yandex is a technology company that builds intelligent products and services powered by machine learning. The company goal is to help consumers and businesses better navigate the online and offline world. Yandex is the largest technology company in the CIS countries.

Yandex Automotive Platform (YAP, Yandex Auto) is a solution based on Android Automotive OS (AAOS) that combines Yandex services (such as Navigation, Alice, Music, Fuel, Parking, Weather, Kinopoisk, etc) and makes them convenient to use in a vehicle.

Yandex Auto provides a full-fledged HMI (Human-Machine Interface) that contains all the necessary components for managing car multimedia system (In-Vehicle Infotainment, IVI) and vehicle parameters (Vehicle Settings, HVAC, etc).

Our solution is the most popular on the Russian market and CIS countries!

I was the main developer and maintainer of Yandex Automotive Platform key system's components:

  • YAP Window System — The special windows composition is the key feature of Yandex Automotive Platform User Interface (YAP UI). It means the all user interactions with IVI performs with consideration of YAP windows composition which implemented in Android. This component is an extension of Android PhoneWindowManager with supporting new windows' types and logic to them layout.
  • YAP Special Signature — A set of changes in Android PackageManager to support second platform signature, which automatically grants specified system permissions to selected HMI components.
  • YAP Dynamic DPI System — This component necessary to change default DPI for selected applications in IVI depending on external conditions. Ultimately, this affects the scale of the application user interface.
  • Head Unit Emulator with all our system and HMI components and SDK Add-On to integrate it into Android Studio.
