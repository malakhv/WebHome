Besides my main job, I have several other pet projects. Below, you could see some of them, and others you would find on my GitHub.

Buzzblender

Buzzblender is an affordable dynamic digital signage software made for Samsung displays, LG Screens and Android devices. The biggest advantages are quick setup, affordable pricing (just five dollars per screen per month), and support for all types of TVs: commercial and regular TVs. In working with multiple customers, one clear trend has been observed: companies that combine QR codes with dynamic promotional rotations typically see engagement and sales improve by around 25%. I am involved in this project as a consultant.

DroidKit

The common library for Android projects. It contains useful utilities and stuff to working with databeses, location, etc. I use this library in all my applications. For more details, please see project page on GitHub.

LogCat

The library project includes classes for writing an app log in Android. Is essentially a wrapper for the standard Android Log class. The main difference is the automatically check the current log's level while trying to write logs. Also, there is a possibility to change the log level for the application at runtime via adb. For more details, please see project page on GitHub.

PascalKit

Pascal Utils Library (PUL) – the set of Pascal units with common userful classes, utilites and tools. For more details, please see project page on GitHub.

S-Dump

S-Dump (Small Dump) is a small program to see and save to external file a dump (like in hex editors) of any file. It useful to investigate file formats and reverse engineering. For more details, please see project page on GitHub.

WebPad

A simple web notepad (WebPad) with text formatting and spell check feature. The WebPad doesn't send the entered text anywhere, but only saves it in a secure local storage. Just try it or see source code on GitHub.
