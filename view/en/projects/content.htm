S-Dump

S-Dump (Small Dump) is a small program to see and save to external file a dump (like in hex editors) of any file. It useful to investigate file formats and reverse engineering. For more details, please see project page on GitHub.

LogCat

The library project includes classes for writing an app log in Android. Is essentially a wrapper for the standard Android Log class. The main difference is the automatically check the current log's level while trying to write logs. Also, there is a possibility to change the log level for the application at runtime via adb. For more details, please see project page on GitHub.

MyLibrary

My personal and private digital library with fiction, non-fiction, self-help, scientific and technical literature. The library includes not only books, but articles, courses, magazines, publishing house catalogues and some collections of books on specific topics. And sure, this project contains software and tools to manage content. Right now, this is a private project, but in the future, I have a plan to publish related software and content engine. The main goal of this project is not collecting as many books as possible, but to collect the best on various topics!

sTranslator

The sTranslator (Small Translator) is a very small application that provides an access to the Yandex Translate service. This application itself determines the language of the original text and can exercise its translation into one of the user-selected language (English, Russian, French, Italian, German, Spanish). At the moment, this project is outdated and is no longer supported.

WebPad

A simple web notepad (WebPad) with text formatting and spell check feature. The WebPad doesn't send the entered text anywhere, but only saves it in a secure local storage. Just try it or see source code on GitHub.
